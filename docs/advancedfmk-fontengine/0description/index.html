矢量字体处理

信息

字体解析部分使用 lwjgl-freetype 完成
不需要额外依赖，Advanced Framework 已将 lwjgl-freetype 静态依赖
该静态依赖将在 Minecraft 1.20.5 + 取消

代码实现

从 ttf 字体到 vtxf​

注意

VTXF 文件已于 v0.4.2 版本中移除

  • 解析字体
  • 获取 SVG 路径
  • 分离多边形
  • 三角剖分 (Earcut)
  • 写入 vtxf 文件
  • 初始化 FreeType 库
  • 从文件或内存加载 ttf 字体
  • 将字体指针存储
  • 通过 FT_Decompose_Outline() 函数提取字符轮廓
  • 将每一个 SVG 组成部分放入队列中
  • 通过队列解析每一步操作，对曲线 (二次/三次贝塞尔曲线) 进行计算，存储入 MultiPolygon 对象
  • 使用 Earcut4j 实现
  • 将所有剖分后的三角形顶点坐标和索引写入文件
